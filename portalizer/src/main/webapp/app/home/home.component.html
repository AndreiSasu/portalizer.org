Welcome to Portalizer!

The project is an open-source Kanban-style list application and aims to be used for sprint retrospective, activity planning and task tracking.

You are logged in as "{{account.firstName}} {{account.lastName}}".
To start, open the boards and feel free to poke around .
Note:
This is a demo version and contains randomly generated boards and users to showcase all features.
The data is wiped out re-generated every day at 23:00 PM GMT.

If you are looking for a Docker container which you can deploy on your local network, have a look at Docker Hub.
